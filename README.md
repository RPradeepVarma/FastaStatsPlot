# Structure plot

The script will parse the fasta sequences in the given fasta file to plot its stats. 

# Note
Script need svgwrite module, numpy and SeqIO

USAGE: python3 FastaStatsPlot.py fasta1.fa fasta2.fa fasta3.fa fasta4.fa

# Input
Multi-fasta files
